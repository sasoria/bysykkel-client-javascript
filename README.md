# bysykkel-client-javascript
Denne løsningen starter en `http-server` på http://localhost:8080, som kan åpnes med en standard web-browser.

## Kjøring
```
$ npm install
$ npm start

Starting up http-server, serving .
Available on:
  http://localhost:8080
```
